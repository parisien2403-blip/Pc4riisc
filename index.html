LA MISSION

Raccourci bureau — mode d'emploi
Vos données restent sur votre ordinateur

TRIGONE PC est conçu pour un usage professionnel en régiment : aucun serveur, aucun cloud, aucun compte. Tout ce que vous saisissez (OMR, validation, réservations ABT) est enregistré uniquement sur ce PC, dans l'espace de stockage du navigateur.

Accès Validation OMR (valideur)

  • La tuile centrale Validation OMR est protégée par un code d'accès réservé aux chefs de bureau.
  • Le code est demandé à la première ouverture, puis mémorisé sur cet ordinateur jusqu'à verrouillage manuel.
  • Code par défaut : communiquez-le en interne au régiment (ex. 4RIISC-VISA) — modifiable côté développement si besoin.
  • Bouton Verrouiller l'accès valideur dans l'app pour exiger le code à nouveau.

Fonctionnement hors ligne

  • L'application tourne sans connexion Internet : une fois ouverte depuis le dossier TRIGONE PC, les trois outils restent utilisables hors réseau.
  • Aucune mise à jour ni synchronisation automatique en arrière-plan.
  • Idéal en cas d'absence de réseau sur site ou en déplacement.

Où sont stockées vos données

  • Brouillons, bibliothèques et archives : sur ce poste (navigateur / disque local).
  • Rien n'est transmis automatiquement vers un serveur externe, une base centralisée ou un tiers.
  • Pas de tracking, pas de télémétrie, pas de connexion obligatoire.

Quand vos données quittent le PC

  • Uniquement si vous le décidez : envoi par e-mail, export PDF, fichier .omr ou pièce jointe que vous choisissez vous-même.
  • TRIGONE ne joint pas de fichier à votre place sans action de votre part.
  • Vous gardez le contrôle de chaque diffusion vers le valideur, l'échelon ou le référent ABT.

Bonnes pratiques

  • Verrouillez votre session Windows lorsque vous quittez le poste.
  • Ne videz pas les données du navigateur sans avoir exporté ou archivé vos dossiers importants.
  • Conservez une copie des PDF ou fichiers .omr envoyés si vous en avez besoin ultérieurement.
  • La responsabilité des données incombe au utilisateur de ce poste, comme pour tout document de travail local.
